# ₿ Bitcoin Price Forecast

This project is an end-to-end **Machine Learning application** that analyzes historical Bitcoin market data and predicts future price movements. It combines **data preprocessing, exploratory data analysis (EDA), feature engineering, interactive visualizations, and predictive modeling** to forecast Bitcoin's **High** and **Low** prices.

The application is built using **Python** and **Streamlit**, allowing users to explore the dataset, compare multiple machine learning models, evaluate their performance, and generate price predictions through an interactive graphical interface.

---

## 🚀 Features

### 📂 Data Processing

* Load and preprocess Bitcoin historical price dataset
* Handle missing values and duplicate records
* Data cleaning and formatting
* Feature selection for model training

### 📊 Exploratory Data Analysis (EDA)

* Dataset overview and descriptive statistics
* Correlation analysis between features
* Distribution plots
* Trend analysis of Bitcoin prices
* Time-series visualizations

### 📈 Data Visualization

* Price trend charts
* Correlation heatmaps
* Scatter plots
* Histograms
* Boxplots
* Interactive Matplotlib visualizations

### 🤖 Machine Learning Models

The project compares the performance of multiple regression algorithms:

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

Each model is trained and evaluated to identify the most accurate predictor for Bitcoin prices.

### 📏 Model Evaluation

Models are evaluated using multiple regression metrics:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score (Coefficient of Determination)

### 💻 Streamlit Application

The project includes an interactive Streamlit interface that allows users to:

* Explore Bitcoin historical data
* Perform EDA interactively
* Train machine learning models
* Compare model performance
* Predict Bitcoin High and Low prices
* Visualize prediction results

## 🎯 Project Objective

The primary objective of this project is to build a complete machine learning pipeline capable of accurately forecasting Bitcoin prices while providing meaningful insights through interactive visualizations. By comparing multiple regression algorithms, the project identifies the model that delivers the best predictive performance for Bitcoin market data.

This project demonstrates practical applications of **data analysis**, **machine learning**, **model evaluation**, and **Streamlit deployment**, making it suitable for data science and machine learning portfolios.
