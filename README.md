<h1 align="center">₿ Bitcoin Price Forecast</h1>

<p align="center">
A <b>Machine Learning & Streamlit Application</b> for analyzing historical Bitcoin data and predicting future <b>High</b> and <b>Low</b> prices using multiple regression models.
</p>

<hr>

<h2>📖 Project Overview</h2>

<p>
This project performs an end-to-end Machine Learning workflow on historical
<b>Bitcoin price data</b>. It includes
<b>data preprocessing</b>,
<b>exploratory data analysis (EDA)</b>,
<b>interactive visualizations</b>,
<b>model training</b>,
<b>model evaluation</b>, and
<b>price prediction</b> using multiple regression algorithms.
</p>

<p>
The application is developed with <b>Python</b> and deployed using
<b>Streamlit</b>, allowing users to interactively explore the dataset,
compare machine learning models, and predict Bitcoin prices.
</p>

<hr>

<h2>🚀 Features</h2>

<h3>📂 Data Processing</h3>

<ul>
<li>Load Bitcoin historical dataset</li>
<li>Handle missing values</li>
<li>Remove duplicate records</li>
<li>Data preprocessing and feature selection</li>
<li>Prepare data for machine learning models</li>
</ul>

<h3>📊 Exploratory Data Analysis (EDA)</h3>

<ul>
<li>Dataset overview</li>
<li>Descriptive statistics</li>
<li>Correlation analysis</li>
<li>Price trend analysis</li>
<li>Distribution analysis</li>
</ul>

<h3>📈 Visualizations</h3>

<ul>
<li>Bitcoin price trends</li>
<li>Correlation heatmaps</li>
<li>Histograms</li>
<li>Scatter plots</li>
<li>Boxplots</li>
<li>Interactive charts using Matplotlib</li>
</ul>

<h3>🤖 Machine Learning Models</h3>

<ul>
<li>Linear Regression</li>
<li>Decision Tree Regressor</li>
<li>Random Forest Regressor</li>
</ul>

<h3>🧾 Model Evaluation</h3>

<ul>
<li>Mean Absolute Error (MAE)</li>
<li>Mean Squared Error (MSE)</li>
<li>Root Mean Squared Error (RMSE)</li>
<li>R² Score</li>
</ul>

<h3>💻 GUI Application</h3>

<ul>
<li>Interactive dashboard using tkinter</li>
<li>Explore historical Bitcoin prices</li>
<li>Train and compare ML models</li>
<li>Predict High and Low prices</li>
<li>Visualize prediction results</li>
</ul>

<h2>🎯 Objective</h2>

<p>
The goal of this project is to build a complete Machine Learning application
that accurately predicts Bitcoin prices while providing meaningful insights
through interactive visualizations. Multiple regression models are compared
to identify the best-performing algorithm for forecasting Bitcoin market prices.
</p>
